# Platform Specifications

Character limits, optimal formats, and posting guidance for each platform.

## Substack (Product Field Notes)

| Spec | Value |
|:-----|:------|
| **Short post** | 1,200-2,000 words |
| **Long post** | 4,000-6,000 words |
| **Title** | Provocative, challenges conventional wisdom. Under 60 chars for email subject. |
| **Structure** | Hook, TLDR, organizing framework, data in prose, real examples, closing reframe |
| **Publish day** | Tuesday or Thursday morning (highest open rates for professional newsletters) |
| **Images** | 1 header image. Inline images for data/charts only. No stock photos. |
| **CTA** | End with question or call to community. "What's your take?" |

### Substack-Specific Rules
- TLDR within the first 3 paragraphs
- Headers are problem-focused: "The Failure Rates Nobody Talks About" not "Section 3"
- One branded term per piece if it fits naturally (e.g., "agentifiable")
- Data woven into prose, not dumped as bullet lists

## LinkedIn

| Spec | Value |
|:-----|:------|
| **Length** | 100-300 words (sweet spot: 150-200) |
| **Character limit** | 3,000 chars (but algorithm favors shorter) |
| **Hook** | First 2 lines must compel "see more" click. Bold claim or contradiction. |
| **Format** | Short paragraphs (1-2 sentences). Line breaks between each. |
| **Hashtags** | Max 3, at the bottom. Relevant to PM/tech audience. |
| **Post day** | Tuesday-Thursday, 8-10am local time |
| **Engagement** | End with a question to drive comments |
| **Images** | Optional. Carousel posts perform well for frameworks/lists. |

### LinkedIn-Specific Rules
- No "I'm excited to announce" openings
- No emoji-heavy formatting (one or two max, if natural)
- Professional but not corporate. Conversational authority.
- Tag people only if genuinely relevant

## X / Twitter

| Spec | Value |
|:-----|:------|
| **Single tweet** | 280 characters |
| **Thread** | 3-7 tweets (sweet spot: 5) |
| **Hook tweet** | Must work standalone. Bold claim, surprising stat, or contrarian take. |
| **Thread format** | 1/ Hook, 2-N/ Supporting points, Final/ Summary + CTA |
| **Post time** | Early morning (7-9am) or lunch (12-1pm) |
| **Images** | Screenshots, charts, or diagrams boost engagement 2-3x |

### X-Specific Rules
- Each tweet in a thread must stand alone (retweets show single tweets)
- No hashtags in threads (looks desperate)
- Quote-tweet your own thread with the hook for repost
- Ratio of value posts to promo: at least 4:1

## TikTok

| Spec | Value |
|:-----|:------|
| **Length** | 30-60 seconds (sweet spot: 45s) |
| **Script format** | Talking head or voiceover with visual cues |
| **Hook** | First 3 seconds must stop the scroll. Surprising stat or bold claim. |
| **Structure** | Hook (3s), Context (10s), Insight (20s), Takeaway (10s), CTA (5s) |
| **Captions** | Always include captions/subtitles |
| **Post time** | 7-9am, 12-3pm, or 7-11pm |

### TikTok Script Format
```
[VISUAL: {what's on screen}]
[HOOK - 3s]: "{Opening line that stops the scroll}"
[CONTEXT - 10s]: "{Set up the problem or situation}"
[INSIGHT - 20s]: "{The core value. What most people get wrong and what's actually true.}"
[TAKEAWAY - 10s]: "{One actionable thing to do differently}"
[CTA - 5s]: "Follow for more PM insights" / "What's your experience?"
```

## Threads

| Spec | Value |
|:-----|:------|
| **Character limit** | 500 chars |
| **Link handling** | Separate link attachment |
| **Hashtags** | None typical |
| **Media** | Images, video |
| **Tone** | Conversational, casual |

### Threads-Specific Rules
- Shorter than LinkedIn, less compressed than X
- Visual-first if possible
- No hashtags needed

## Bluesky

| Spec | Value |
|:-----|:------|
| **Character limit** | 300 chars |
| **Link handling** | Via facets (rich text) |
| **Hashtags** | None (use feeds for topic targeting) |
| **Media** | Images |
| **Tone** | Direct, community-oriented |

### Bluesky-Specific Rules
- Concise due to 300 char limit
- Use feeds/lists for topic targeting instead of hashtags
- Community-oriented tone

## Cross-Platform Adaptation Flow

```
Substack (canonical) → LinkedIn (condense to 1 insight) → X (compress to hook + 3 points) → TikTok (script the hook)
```

Or reverse:
```
X thread (test the hook) → LinkedIn (expand with context) → Substack (full treatment if engagement was high)
```
